# ProjectX-backend

## 1. Installing Dev dependencies:

```bash
npm install
```

## 2. Used scripts:
Run the development server
```bash
npm run dev
```

Compile TS files and Start the production Server
```bash
npm run prod
```

Compile TS files
```bash
npm run build
```

Start the production server
```bash
npm run start
```
